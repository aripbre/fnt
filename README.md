# fnt

## 安装

### [富能通服务平台](http://service.funenc.com/)

```sh
# 选择完免登授权类型之后填写对应的参数
# 参数在富能通服务平台上
$ npm install fnt -g
```

![avatar](http://onx1jjc4v.bkt.clouddn.com/fnt.png)

## 运行
```sh
$ fnt create test
$ cd test && npm install && npm run dev
$ cd client && npm install && npm run dev
```
